# **App Name**: SUBI

## Core Features:

- AI Chat Interface: A responsive, real-time chat interface for seamless interaction with the SUBI AI assistant, including a user input bar.
- Generative AI Responses: SUBI will use a generative AI tool to craft intelligent, context-aware, and helpful textual responses to user queries.
- Dynamic Chat History Display: Displays the chronological flow of user messages and SUBI's responses in an elegant, scrollable chat feed.
- Collapsible Sidebar Navigation: A smooth, animated left-hand navigation panel featuring a 'Home' link, designed to be expanded or collapsed to maximize screen space.
- Adaptive Layout: Ensures the chat interface and navigation adjust fluidly across various device sizes and orientations for an optimal viewing experience.

## Style Guidelines:

- Primary Color: A refreshing, modern blue (#66A8EB), chosen for its sleek and calming digital presence, providing contrast against light backgrounds.
- Background Color: A very subtle, cool-toned white (#EFF3F5), highly desaturated and bright, creating a 'glassy' and expansive canvas for the application content.
- Accent Color: A vibrant Claude's orange (#FF8000), specifically chosen per user request to provide energetic highlights for interactive elements and calls to action.
- Main font: 'Inter' (sans-serif), selected for its contemporary, legible, and objective feel, perfectly suited for both headlines and the conversational text within the chat interface.
- Embrace a minimalist, line-based icon style with soft curves to maintain the app's 'light, sleek, and glassy' aesthetic and ensure clarity.
- Utilize generous whitespace, translucent 'frosted glass' card elements for content containers, and clean, defined edges to contribute to a sleek and expansive feel.
- Implement smooth, 'sexy' transitions and micro-interactions for all UI elements, including navigation collapse, chat message entry, and hover states, to ensure a fluid and engaging user experience.
